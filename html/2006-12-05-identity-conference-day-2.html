(OASIS subgroup) Andy from OOTAO is showing an example of Kintera, a non profit services provider, sharing data with Le leche a non profit. XDI is the interface that Le Leche uses to access the Kintera systems. It lets a machine make requests to a remote system kind of like SQL lets an app query a database. Kind of interesting, and useful for large enterprise systems. The example being shown requires that the two parties know each other and setup a complex contract before interactions are allowed to occur. Useful for those that need it, but so far it seems rather heavy for main stream consumption.
